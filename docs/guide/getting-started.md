# Getting Started

Four Tailwind UI is a collection of Vue 3 components built with Tailwind CSS.

## Features

- **Vue 3 Composition API** - Modern Vue development
- **TypeScript** - Full type support
- **Tailwind CSS** - Utility-first styling
- **Dark Mode** - Built-in theme switching
- **Accessible** - ARIA-compliant components

## Quick Start

```bash
npm install @four-bytes/four-tailwind-ui
```

```ts
// main.ts
import '@four-bytes/four-tailwind-ui/style.css'
```

```vue
<script setup>
import { Button } from '@four-bytes/four-tailwind-ui'
</script>

<template>
  <Button variant="primary">Click me</Button>
</template>
```
